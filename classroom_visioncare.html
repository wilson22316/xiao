中、老年人視力保健問題

中老年人為什麼需要戴眼鏡

根據醫學統計，一般東方人在超過40歲以後，由於身體各器官已逐漸呈現衰退現象，眼睛也因為水晶體彈性衰退，調節視力的能力不足，而產生初期看近物會感到吃力，後期即看不清楚近物，而形成老花，如果您有下列徵狀，表示您需要戴老花眼鏡或漸進多焦點鏡片，請稍加留意。

  1. 看報拿得遠遠的才能看得清楚。
  2. 白天視力比晚間好，光線不足時覺得吃力。
  3. 近處看細小的字覺得吃力。
  4. 閱讀時，眼睛容易疲勞或掉淚，並有肩膀硬、頭痛、噁心眩暈的症狀。
  5. 閱讀不久後，發現視力模糊不清。

重視40歲以後的視力

一般人步入40歲後，視力會逐漸衰退，且易患眼睛疾病，以內障類較多。內障疾病可分三種，有綠內障(青光眼)、黑內障、白內障，其中老年人最容易罹患的就是白內障。

發生現象

初期水晶體會有異狀，眼睛很容易勞，眼睛並會產生飛蚊症狀，就是閉上一隻眼睛看東西會有物像重疊的現象，然後，眼睛漸漸模糊，老花眼度數減低，以致於無法配到合適的老花眼鏡，這種現象由初期到形成快則1~3年，慢則5~6年，同時要注意糖尿病、睫狀體發炎等等併發症。
因此，一到中年，如有眼睛流淚、眼睛模糊、疼痛或頭痛、血厭高等情形，應定期接受眼科醫師檢查。我們已知40歲以後的視力已逐漸衰退，因此，在生活上及工作上要注意：

  1. 增加照明度，請用燭光較大的燈泡，尤其是書桌上的閱讀，光線更需充足。
  2. 開車時，保持適當的速度並稍作休息。
  3. 避免烈陽暴曬，請戴上太陽眼鏡。
  4. 防止眼睛疲勞，請多休息。

防止視力老化的保健方法

  1. 常常上下左右地轉動眼珠，並多眨眨眼。
  2. 在某特定點上，集中注視。
  3. 所有的閱讀或近處工作，一定得要與眼睛保持14吋(學36.5cm)的距離，同時要常常眨眼和集中注視每個字上。
  4. 一天需花半個小時以上做以上各項訓練。
  5. 如老花視力突然好轉，度數減輕並非正常現象，可能是眼疾前兆，速找醫師診治。

防止近視的保健方式

營養均衡

  1. 適當攝取五大營養，含糖高的食物少吃。
  2. 多吃新鮮蔬菜，不可偏食。

姿勢端正

  1. 看書時保持30公分的距離。
  2. 不可躺著看書。
  3. 讀書或工作，請30分鐘休息5分鐘。
  4. 選擇印刷良好，字體適中的書籍，並使用較深顏色的鉛筆。
  5. 定期調整桌椅高度，桌子必須與手肘同高，雙腳必須踏到地板。

收看電視注意事項

  1. 保持兩公尺以上或畫面對角線保持6~8倍距離。
  2. 以不超過一小時為準，最好每30分鐘休息10分鐘。
  3. 室內光線要充足，不可明暗相差太大。
  4. 不可躺著看電視。螢幕稍低於水平視線，且不要太偏角度，最好在30度內。

注意採光

  1. 眼睛最舒適最清楚的亮度是40~50瓦特的電燈或20瓦日光燈。
  2. 請常擦拭燈管、燈泡，以免灰塵影響照明度。

其它注意事項

  1. 充分睡眠及戶外活動。時常集中精神眺望遠方目標。
  2. 每3~6個月或一年兩次視力檢查。
  3. 近視400度以下者，只在上課或平常時戴，長時間閱讀不必載，500度以上者最好能有兩副眼鏡，閱讀時可使用度數較淺的眼鏡。漸近多焦鏡片是一副眼鏡同時適合遠、中、近距離的最好選擇。

早日發現近視的各種現象與方法

  1. 頭往前傾看遠處，或看遠物時會瞇著眼睛。
  2. 眼睛與書本過近。
  3. 不能集中精神讀書、做功課，學業退步。
  4. 在近處看電視。
  5. 頸項側彎，側著看東西。
  6. 經常揉眼睛、有畏光情形。
  7. 常抱怨眼睛累或頭痛。

自己測驗是否近視了？

  1. 望遠發現法從固定的地方，決定視力所及的遠方目標，經常看它以測定自己視力狀態。
  2. 明暗發現法：
    • 隨雨天或黃昏時，視線會較差，如果有近視傾向者，視力會大幅度降低。
  3. 小孔發現法：
    • 視力正常者無法透過小孔看清物體，反之，近視者能透過小孔將遠方細小目標看得清楚。
  4. 眼睛疲勞發現法：
    • 剛開始近視者無法持續閱讀一小時，如果視力惡化、眼睛會疲勞、模糊、眼瞼澀重。
  5. 單眼發現法：
    • 用單眼交互看月曆或視力檢查表(距離約3~4公尺)。如自己感覺有近視現象時，應由專業眼鏡公司為您複檢。

近視如何矯正？

  1. 凹透鏡型的鏡片可以使光線更精確的落在視網膜上，獲得更清晰的影像。
  2. 依近視的程度而決定是否需要配戴框架眼鏡或隱形眼鏡，以保持清晰的視力。
  3. 如果需要配戴矯正用的隱形眼鏡或框架眼鏡，可能需要適應一下配戴後的生活。
